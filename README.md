# 🎓 Student Management System

A **Java console-based application** to manage student records using **Object-Oriented Programming (OOP)** concepts.

---

## ✨ Features
- ➕ Add new students  
- 📋 Display student details  
- ✏️ Update student information  
- ❌ Delete student records  

---

## 🛠️ Technologies Used
- **Language:** Java  
- **Concepts:** OOP, Classes & Objects, Inheritance  
- **Tools:** VS Code / IntelliJ IDEA  

---

## 📖 How It Works
1. Run the program in your IDE or terminal.  
2. Choose an option from the menu (Add / View / Update / Delete).  
3. The system performs the requested operation on the student data.  

---

## 🚀 Future Enhancements
- Add file handling to save data permanently  
- Add a GUI using JavaFX or Swing
